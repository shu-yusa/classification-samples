# Classification samples
Learning machine learning algorithms using `scikit-learn`.

## Requirements
* python3.6.1
* virtualenv

## Setup
Create `virtualenv` environment by
```bash
virtualenv env
source env/bin/activate
```

Install modules and libraries by
```bash
pip install -r requirements.txt
```

## Execute
Execute script
```bash
python classify.py
```
